How to use the API?

  1. To use the API, go to
    https://bookdatabase-djangorest.herokuapp.com/api/

  2. To sign up through the API, go to
    https://bookdatabase-djangorest.herokuapp.com/api/profile/

  3. To log in to the API, go to
    https://bookdatabase-djangorest.herokuapp.com/api/login/
    You'll get a special token which needs to be passed in the HTML header,
    e.g. through ModHeaders (you can download a ModHeader extension for your browser).

  4. To see or add books, go to
    https://bookdatabase-djangorest.herokuapp.com/api/books/
    You need to provide the token to add a book (see point 3).

  5. To see or add authors, go to
    https://bookdatabase-djangorest.herokuapp.com/api/authors/
    You need to provide the token to add an author (see point 3).

  6. To see or add publishers, go to
    https://bookdatabase-djangorest.herokuapp.com/api/publishers/
    You need to provide the token to add a publisher (see point 3).

  7. To see, edit or delete a particular item, add its ID at the end of the link,
    e.g. https://bookdatabase-djangorest.herokuapp.com/api/books/1/
    To edit or delete the given item, you need to provide the token (see point 3)
    and be the owner of the item.

  8. To see registered users, go to
    https://bookdatabase-djangorest.herokuapp.com/api/profile/
